# Picksy
Picksy is a WordPress-based web application that enables users to submit content through frontend forms and view submitted posts dynamically. 

The project focuses on creating a complete user-generated content workflow using WordPress tools and plugins, without requiring users to access the admin dashboard.

This repository showcases my practical experience in building and configuring WordPress websites using real-world workflows.

---

## 🎯 Project Overview

Picksy demonstrates how WordPress can be used to:
- Collect user-generated content from the frontend
- Manage submissions without exposing the admin panel
- Display submitted content dynamically
- Build and deploy a functional website using commonly used WordPress tools

---

## 🔧 Tools & Technologies

- WordPress (CMS)
- Astra Theme
- Elementor
- WP User Frontend

---

## ✨ Key Features

- Frontend user submission forms
- User registration and login flow
- Dynamic display of submitted posts
- Responsive layouts built with Elementor
- Plugin-based workflow configuration

## 🚀 Running the Project Locally

This project was developed and tested using **Local WP** for local WordPress development.

To explore or recreate a similar setup:
- Set up a local WordPress site using Local WP
- Activate the Astra theme
- Install and configure Elementor and WP User Frontend
- Create frontend submission forms and configure display pages

> Themes and plugins are not included in this repository due to licensing restrictions.
